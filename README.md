# Customer_Shopping_Behavior_Analsis
Customer Shopping Behavior Analysis using Python, SQL, and Power BI. Includes RFM segmentation, statistical tests, and interactive dashboards for actionable business insights.

Overview
Analyze customer shopping behavior to identify trends, optimize marketing, and improve sales, loyalty, and engagement.

Key Features
Data Cleaning & Feature Engineering in Python
RFM Analysis & Customer Segmentation (Champions, Big Spenders, Loyal, Potential Loyalists, At Risk)
Statistical Analysis (T-test for subscribers vs non-subscribers)
SQL Queries for Revenue, Product, and Customer Insights
Interactive Power BI Dashboard with filters for subscription, age, category, and shipping type

Dataset
3,900 purchases across 18 columns
Includes demographics, purchase details, shopping behavior, and reviews

Insights & Recommendations
Big Spenders lead revenue; Potential Loyalists = growth opportunity
Loyal Customers are ideal for retention and upselling
Optimize discount strategy and product positioning
Target marketing on high-revenue age groups

Technologies Used
Python, Pandas, NumPy, PostgreSQL, Power BI
